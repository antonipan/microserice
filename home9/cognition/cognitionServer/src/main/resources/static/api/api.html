Api

Ниже представлен api приложения "cognition".

/api

@GET - получить api по текущему приложению

TEACHER

/teachers

@GET - добавляет список учителей в базу данных
@POST - показывает список учителей из базы данных

/teachers/{id}

@GET - показывает список учителей из базы данных
@POST - добавляет список учителей в базу данных
@PUT - обновляет данные об учителе по его id в базе данных
@DELETE - удаляет учителя по его id из базы данных

/teachers/sort/{paramSorted}

@GET - если paramSorted = name - происходит, сортировка по имени; если "subject" - по предмету учителя; если "experience" - по опыту

/teachers/filter/{experience}

@GET - фильтрует и отображает список учителей по стажу работы

/teachers/subjects/{subject}

@GET - фильтрует и отображает список учителей по стажу работы

QUESTIONNAIRE

"/questionnaires"

@GET - добавляет список методик в базу данных
@POST - показывает список методик из базы данных

"/questionnaires/{id}"

@GET - получает методику по id из базы данных
